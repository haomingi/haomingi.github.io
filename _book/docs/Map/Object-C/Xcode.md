# Xcode

## Xcode简介

* **Xcode是苹果的集成开发环境（IDE）**

* 您可以使用Xcode为Apple产品构建应用程序，包括iPad，iPhone，Apple Watch，Apple TV和Mac

* Xcode提供了一些工具来管理整个开发工作流程 - 从创建应用程序到测试，优化并提交给App Store

## Xcode要求
* 必须是macOS系统
* 推荐App Store下载（XcodeGhost事件造成的影响：https://www.oschina.net/news/66349/xcodehost-roundup）
